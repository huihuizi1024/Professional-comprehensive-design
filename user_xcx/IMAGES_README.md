# 快递柜小程序图标说明

## tabBar 图标要求

小程序需要以下8个图标文件（每个81x81像素，PNG格式）：

### 快递柜图标
- `images/cabinet.png` - 未选中状态
- `images/cabinet-active.png` - 选中状态

### 取件图标
- `images/pickup.png` - 未选中状态
- `images/pickup-active.png` - 选中状态

### 订单图标
- `images/order.png` - 未选中状态
- `images/order-active.png` - 选中状态

### 个人中心图标
- `images/profile.png` - 未选中状态
- `images/profile-active.png` - 选中状态

## 图标设计建议

- 尺寸：81x81 像素
- 格式：PNG
- 未选中图标：灰色 (#999999)
- 选中图标：紫色 (#667eea)
- 背景透明

## 临时解决方案

如果暂时没有图标，可以：
1. 使用在线图标库（如 iconfont）下载合适的图标
2. 使用设计工具（如 Figma、Sketch）创建简单图标
3. 从微信小程序官方示例中复制图标

## 图标放置位置

将图标文件放置在 `user_xcx/images/` 目录下。
